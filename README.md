🚀 Project Overview
This webpage was created as part of a technical task to demonstrate proficiency in web fundamentals. It strikes a balance between professional work history and personal motivation, showcasing a clean, modern aesthetic.

Key Features
Semantic HTML: Structured with accessibility and SEO in mind.

Responsive Design: Optimized for viewing on desktops, tablets, and mobile devices.

Clean Styling: Built with a custom CSS color palette and typography from Google Fonts.

Skill Highlight: A grid layout showcasing a mix of soft skills (Communication, Public Relations) and technical competencies (Git, GitHub).

🛠️ Technology Stack
HTML5: Structure and content.

CSS3: Custom styling using CSS Grid and Flexbox for a 2-column feel on wider screens.

Google Fonts: Utilizing the 'Inter' typeface for high readability.

GitHub Pages: Deployed and hosted as a live static site.
